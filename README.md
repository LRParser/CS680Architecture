# CS680Architecture
CS680Architecture

## Installation

pip install flask

## Running the client

Open a terminal and do:

```
cd LectureProClient
export FLASK_APP=client.py
export FLASK_ENV=development
flask run
```

### Running the server


Open another terminal and do:

```
cd LectureProClient
export FLASK_APP=server.py
export FLASK_ENV=development
flask run
```
